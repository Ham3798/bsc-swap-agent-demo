# Read-Only Planner Web App

This app is the first web consumer of `@bsc-swap-agent-demo/core`.

Current scope:

- natural-language plan input
- MetaMask address connect
- follow-up question and answer flow
- plan summary and typed decision trace
- route, payload, submission, and raw JSON panels

Not included yet:

- actual sign/send
- public transaction handoff button
- private relay execution
- production Vercel hardening
